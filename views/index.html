Timestamp Microservice

Convert between Unix timestamps and UTC dates

API Usage

GET /api/<date>

The date parameter can be:

  • A valid date string (e.g., "2023-07-15")
  • A Unix timestamp in milliseconds (e.g., 1689379200000)
  • Empty to return the current time

Example API Responses

Valid Date String

GET /api/2015-12-25

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Unix Timestamp

GET /api/1451001600000

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Invalid Date

GET /api/invalid-date

{
  "error": "Invalid Date"
}
